How to Download Spatial_v2

  • Step 1:  download this zip file to a directory (folder) of your choice.
  • Step 2:  unzip it.  This will create a directory called spatial_v2 containing the complete package.  (Some browsers will allow you to perform steps 1 and 2 as a single operation.)
  • Step 3:  edit the file spatial_v2/startup.m, following the instructions in that file.  Only a single line needs to be changed.
  • Step 4 (optional):  spatial_v2 includes a local copy of the documentation, which is identical to the copy on the Web.  If you would like to use this local copy, then either bookmark or create a shortcut to the file spatial_v2/documentation/index.html.
Don't forget to read the terms and conditions under which spatial_v2 is made available to you.
